# 🧩 Customer Behavior Dashboard (SQL-Python-PowerBI)

## 📖 Project Overview

An interactive Customer Behavior Dashboard built in Power BI to analyze customer purchase patterns, subscription trends, and segment performance.
The project uncovers actionable insights that help improve customer retention, optimize marketing, and boost overall sales performance.

**🔁 Project Workflow**
- 🧭 Step-by-Step Process

- 1️⃣ Business Problem Statement – Identify customer insights goals
- 2️⃣ Data Modelling & EDA in Python – Clean and prepare data
- 3️⃣ Data Analysis in SQL – Derive metrics and KPIs
- 4️⃣ Dashboard Creation in Power BI – Build interactive visuals
- 5️⃣ Project Report – Summarize findings
- 6️⃣ Presentation using Gamma AI – Present to stakeholders
- 7️⃣ GitHub Repository – Upload final .pbix, visuals & documentation

## ⚙️ Data Preparation

- ✅ Cleaned null and duplicate values
- ✅ Standardized columns (Category, AgeGroup, Gender)
- ✅ Created DAX measures – Total Revenue, Avg Purchase Value, Avg Rating
- ✅ Built data relationships to enable cross-filtering across segments

## 📊 Dashboard Overview

The dashboard provides a snapshot of key business metrics such as total revenue, average purchase value, and customer satisfaction.
Visual filters for Subscription Status, Gender, and Product Category allow easy exploration.

## 📈 Key Insights

- 🔹 Total Revenue: $233K from 3.9K customers
- 🔹 Average Purchase Value: $59.8 | Avg Rating: 3.75
- 🔹 Subscription: Only 27% subscribed but higher spenders
- 🔹 Top Category: Clothing leads, followed by Accessories
- 🔹 Customer Segment: Loyal customers generate the highest revenue
- 🔹 Age Group: Middle-age and Adults dominate revenue contribution

## 💡 Business Recommendations

- ✅ Improve customer satisfaction — target rating >4.0
- ✅ Offer exclusive benefits for subscribers (10% discount, free shipping)
- ✅ Bundle footwear with accessories to increase average order value
- ✅ Launch tiered loyalty programs (Silver, Gold, Platinum)
- ✅ Focus marketing on high-spending segments (Adults & Middle-Age)
